# Design references (Figma exports)

Drop **PNG / WebP** screen exports or **Copy as PNG** from Figma here so you can `@`-mention them in Cursor or attach to chat when asking to match a layout.

Suggested names: `hub.png`, `club-tab.png`, `match-day.png`, etc.

Optional: keep files small (≤ ~1–2 MB each) so the repo stays fast to clone.
