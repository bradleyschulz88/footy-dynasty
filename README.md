# Footy Dynasty

Australian rules football manager — climb from community leagues to the AFL.

- **Live app:** [footy-dynasty.vercel.app](https://footy-dynasty.vercel.app)
- **Source:** [`footy-dynasty/`](footy-dynasty/) (React + Vite)
- **Docs:** [`footy-dynasty/README.md`](footy-dynasty/README.md)

## Quick start

```bash
cd footy-dynasty
npm install
npm run dev
```

## Scripts (from `footy-dynasty/`)

| Command | Purpose |
|---------|---------|
| `npm test` | Unit tests (Vitest) |
| `npm run build` | Production build |
| `npm run lint` | ESLint |
| `npm run test:e2e` | Playwright smoke tests |

CI runs on every push to `main` (see [`.github/workflows/ci.yml`](.github/workflows/ci.yml)).
